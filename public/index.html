Project brief

Web Vulnerability Scanner

This project scans authorized web targets and aggregates findings from multiple security modules into a unified analysis report.

Use the dashboard to review prioritized risks, inspect technical evidence, and download a PDF report for tracking and remediation.

Input Submit one or more target URLs from the intake panel.
Scan modules Recon, vulnerability checks, token analysis, secret discovery, and brute-force evaluation.
Analysis report AI summary, severity signals, and categorized findings per target.
Best use Run only against authorized targets and review findings before acting on them.
